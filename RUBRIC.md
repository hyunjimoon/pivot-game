# 📐자 — 강연이 질문에 답하는가

> 이 강연이 [11개 사전질문](QUESTIONS.md)을 얼마나 *계산끝내* 답하는지 재는 자(尺). 질문당 0–3점, 만점 33.

## 척도

| 점수 | 상태 | 판정 |
|:--:|:--|:--|
| **0** | 부재 | 언급조차 없음 |
| **1** | 언급 | 주제만 스침, 답 없음 |
| **2** | 응답 | 답은 있으나 추상적·구체사례 없음 |
| **3** | 계산끝낸 응답 | 답 + 숫자/일화 1개 + "이건 내 얘기" 공명 |
